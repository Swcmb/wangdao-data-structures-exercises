# 王道数据结构与算法实现

## 项目简介

本项目基于王道考研《数据结构与算法》笔记，实现了书中所有章节的C语言代码。项目涵盖了从基础线性表到高级排序算法的完整实现，旨在为学习数据结构与算法的同学提供清晰、易懂的参考代码。

## 项目特点

- **完整覆盖**：实现了王道考研数据结构与算法书中的所有核心内容
- **代码清晰**：采用简洁明了的代码风格，便于理解和学习
- **结构合理**：按照章节组织代码，层次分明
- **注释完善**：关键部分配有详细注释，帮助理解算法原理

## 各章节详细内容

### 第二章 线性表
- **顺序表**：实现了静态分配和动态分配两种方式，包括初始化、插入、删除、查找等基本操作
- **单链表**：实现了带头结点和不带头结点两种形式，支持增删改查操作
- **双链表**：实现了带头结点的双链表，支持双向遍历和操作
- **循环链表**：实现了循环单链表和循环双链表，解决了链表首尾相连的问题
- **静态链表**：使用数组模拟链表结构，无需指针操作

### 第三章 栈和队列
- **顺序栈**：基于数组实现的栈结构，支持入栈、出栈、判空等操作
- **链式栈**：基于链表实现的栈结构，动态分配内存
- **循环队列**：解决了顺序队列的假溢出问题，实现了高效的队列操作
- **链式队列**：基于链表实现的队列，支持动态扩容
- **双端队列**：两端都可以进行插入和删除操作的特殊队列
- **栈的应用**：实现了括号匹配等经典应用

### 第四章 串
- **串的存储**：使用静态数组实现串的存储
- **朴素模式匹配**：实现了简单的模式匹配算法
- **KMP算法**：实现了优化的模式匹配算法，包括next数组和nextval数组的构建

### 第五章 树
- **二叉树**：基于链式存储结构实现的二叉树
- **遍历算法**：实现了先序、中序、后序和层序遍历
- **线索二叉树**：实现了二叉树的线索化，提高遍历效率
- **二叉排序树**：实现了BST的插入、删除和查找操作
- **平衡二叉树**：实现了AVL树的旋转和平衡操作
- **哈夫曼树**：实现了哈夫曼编码算法

### 第六章 图
- **邻接矩阵**：使用二维数组实现图的存储
- **邻接表**：使用链表实现图的存储，节省空间
- **遍历算法**：实现了深度优先遍历(DFS)和广度优先遍历(BFS)
- **最小生成树**：实现了Prim算法和Kruskal算法
- **最短路径**：实现了Dijkstra算法和Floyd算法
- **拓扑排序**：实现了AOV网的拓扑排序算法
- **关键路径**：实现了AOE网的关键路径算法

### 第七章 查找
- **顺序查找**：实现了线性查找算法
- **折半查找**：实现了二分查找算法，要求数据有序
- **分块查找**：实现了索引顺序查找，结合了顺序查找和折半查找的优点
- **红黑树**：实现了自平衡二叉查找树，保证O(log n)的查找时间复杂度
- **B树**：实现了多路平衡查找树，适合外存访问
- **散列查找**：实现了哈希表查找，包括哈希函数设计和冲突处理

### 第八章 排序
- **插入排序**：实现了直接插入排序、折半插入排序和希尔排序
- **交换排序**：实现了冒泡排序和快速排序
- **选择排序**：实现了简单选择排序和堆排序
- **归并排序**：实现了2路归并排序
- **基数排序**：实现了桶排序算法

## 编译和运行

所有代码均使用C语言编写，可以在任何支持C语言编译器的环境中编译运行。

### 编译方法

```bash
# 编译单个文件
gcc filename.c -o filename

# 编译并运行
gcc filename.c -o filename && ./filename
```

### 示例

```bash
# 编译顺序表静态分配代码
gcc chapter2/seq_list_static.c -o seq_list_static

# 运行
./seq_list_static
```
