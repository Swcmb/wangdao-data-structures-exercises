项目简介

本项目基于王道考研《数据结构与算法》笔记，实现了书中所有章节的 C 语言代码。 项目涵盖了从基础线性表到高级排序算法的完整实现，旨在为学习数据结构与算法的同学提供清晰、易懂的参考代码。

编译方法 
# 编译单个文件
gcc filename.c -o filename

# 编译并运行
gcc filename.c -o filename && ./filename

# 示例：编译顺序表静态分配代码
gcc chapter2/seq_list_static.c -o seq_list_static
./seq_list_static
